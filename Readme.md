

Forntend: Angular 17

**Requirments:**
1. Node: v14.21.3
2. Angular CLI: 17.0.2
   ```npm install -g @angular/cli@17```
3. npm: 6.14.18

**Steps to run the project:**
1. Clone the project
   ``
git clone https://github.com/bdshanto/personal-notepad
``
2. Go to the project directory
   ```cd src/frontend```
3. Run the following command
   ```npm install```
